## ESP32 Basic Series Node – Small Mimic Panel Controller

This ESP32-based node is designed for compact mimic panels in model railway layouts. It provides intuitive route control and comprehensive feedback through LED indicators.
Key Features

- 5 Selectable Routes:
Routes can be triggered either by momentary push buttons on the panel or by consumed events received from another node on the network.
- 8 Turnout Control Outputs:
Controls up to 8 separate turnouts with configurable staggered delays between activation events, ensuring realistic and reliable operation.
- 16 LED Output Channels:
Dedicated outputs for mimic panel indicators, providing clear visual feedback of route status, turnout positions, and track occupancy.
- Standard CAN Interface:
Uses the widely adopted SN65HVD230 transceiver module, fully compatible with the rest of the Basic Series nodes available in the repositories.

This node is ideal for small to medium mimic panels where space is limited but full route control and clear indication are required. It maintains consistency with the existing Basic Series ecosystem for easy integration into larger layouts.
